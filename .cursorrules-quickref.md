# .cursorrules Quick Reference Sheet + Memory Bank Tags Cheat Sheet

## 🧠 Mindset
- Always use the **Scratchpad** to plan & track.
- Always use the **Memory Bank** to save lessons and templates.
- Always learn from mistakes — document them immediately.

---

## ✅ New Task Workflow

1. **Review** Scratchpad
2. **Clear** unrelated old content
3. **Summarize** new task
4. **Plan steps** with todo markers:
    ```
    [X] Step 1 done
    [ ] Step 2 next
    ```
5. **Search Memory Bank** for relevant examples:
    ```bash
    venv/bin/python3 tools/memory.py search --query "keywords"
    ```
6. **Execute** task
7. **Update Scratchpad** after milestones

---

## 🧹 After Task Completion

- **Reflect** in Scratchpad
- **Save to Memory Bank**:
    ```bash
    venv/bin/python3 tools/memory.py save --title "Task Title" --tags tag1,tag2 --notes "Important lessons, fixes, templates"
    ```

---

## 🛠️ Key Tools

- **Screenshot + Verify**:
    ```bash
    venv/bin/python3 tools/screenshot_utils.py URL --output output.png
    venv/bin/python3 tools/llm_api.py --prompt "Describe page" --provider openai --image output.png
    ```

- **LLM API Query**:
    ```bash
    venv/bin/python3 ./tools/llm_api.py --prompt "Your question" --provider "openai"
    ```

- **Web Scrape**:
    ```bash
    venv/bin/python3 tools/web_scraper.py --max-concurrent 3 URL1 URL2
    ```

- **Web Search**:
    ```bash
    venv/bin/python3 tools/search_engine.py "search keywords"
    ```

---

## 📋 Lessons to Always Remember

- Activate `./venv` first — use `uv pip install` if available.
- Multiline git commits:
    ```bash
    git commit -F message.txt
    ```
    (then delete `message.txt`)
- Handle UTF-8 encoding properly.
- Use `'seaborn-v0_8'` for matplotlib.
- Prefer `gpt-4o` model for OpenAI API.
- Prioritize "recent" or "2025" for news searches.
- Memory Bank is your **secret weapon** — grow it relentlessly.

# 🏷️ Memory Bank Starter Tags Cheat Sheet

Use these **standard tags** to keep your Memory Bank clean and powerful:

| Tag Group           | Example Usage               | Purpose                                   |
|---------------------|------------------------------|-------------------------------------------|
| `topic-[subject]`    | `topic-python`, `topic-llm`   | Fast search by subject or tool            |
| `task-[type]`        | `task-debugging`, `task-setup`| Search by type of task                    |
| `fix-[issue]`        | `fix-importerror`, `fix-api`  | Quickly retrieve fixes for common errors |
| `template-[usecase]` | `template-llm-prompt`, `template-email` | Find reusable templates         |
| `workflow-[process]` | `workflow-scrape-verify`, `workflow-train-llm` | Retrieve full step-by-step flows |

---

## 🏷️ Example Tagging When Saving Memories

Save memory like this:
```bash
venv/bin/python3 tools/memory.py save --title "Fix Screenshot Width Error" --tags fix-screenshot tool-screenshot_utils topic-python --notes "Resolved an issue with specifying screenshot width in screenshot_utils.py"
```

Mix and match **multiple tags** for even sharper future search!
